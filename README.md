🚀 QWERT.ai

QWERT.ai is a full-stack AI-powered web application that helps users create high-quality content using multiple AI tools, including article generation, blog title generation, image generation, background removal, object removal, and AI-powered resume review.

🌐 Live Demo

🔗 https://qwert-ai-7pug.vercel.app

✨ Features

- 🤖 AI Article Generator
- 📝 AI Blog Title Generator
- 🎨 AI Image Generator
- 🖼️ AI Background Remover
- ✂️ AI Object Remover
- 📄 AI Resume Review (ATS Score & Suggestions)
- 👥 Community Image Sharing
- 🔐 Secure Authentication with Clerk
- 💎 Premium Features

🛠️ Tech Stack

Frontend
- React.js
- Vite
- Tailwind CSS
- Clerk Authentication

Backend
- Node.js
- Express.js
- PostgreSQL (Neon)
- OpenAI API
- Cloudinary

🚀 Deployment

- Frontend: Vercel
- Backend: Vercel
